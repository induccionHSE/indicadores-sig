# Sistema de Indicadores SIG
 
Matriz estructurada de indicadores del Sistema Integrado de Gestión (SIG) con tablero ejecutivo en Looker Studio. Centraliza el seguimiento estratégico de indicadores de calidad, ambiente y seguridad en una sola fuente de verdad.
 
> **v1.0 — Producción**
 
---
 
## Funcionalidades
 
| Módulo | Descripción |
|---|---|
| **Matriz de indicadores** | Estructura en Google Sheets con indicadores por proceso, meta, frecuencia y responsable |
| **Registro periódico** | Captura de datos de indicadores por período |
| **Dashboard ejecutivo** | Tablero Looker Studio con tendencias, cumplimiento y alertas visuales |
| **Alertas de cumplimiento** | Identificación automática de indicadores por debajo de meta |
 
---
 
## Stack
 
- **Base de datos:** Google Sheets
- **Visualización:** Looker Studio
- **Automatización:** Google Apps Script
- **Modelo de referencia:** Enfoque institucional (UNAD)
